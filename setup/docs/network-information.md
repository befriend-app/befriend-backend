# Network Information

These keys are used for identifying your network. A unique network token is automatically generated when your server runs for the first time.

https://github.com/user-attachments/assets/cb3ae155-7707-49ad-a760-967f0f741539

## Environment Variables

Configure the following environment variables in your `.env` file:

```
ADMIN_NAME=""
ADMIN_EMAIL=
NETWORK_NAME="<YOUR_NETWORK_NAME>"
NETWORK_API_DOMAIN=<YOUR_API_DOMAIN>
NETWORK_LOGO=<YOUR_LOGO_URL>
NETWORK_APP_ICON=<YOUR_APP_ICON_URL>
```

## Variable Descriptions

- **ADMIN_NAME**: The name of the network administrator
- **ADMIN_EMAIL**: Email address for the network administrator
- **NETWORK_NAME**: Display name for your brand on the Befriend network
- **NETWORK_API_DOMAIN**: The domain where your API is hosted
- **NETWORK_LOGO**: URL to your network's logo image
- **NETWORK_APP_ICON**: URL to your network's app icon image

Make sure to replace the placeholder values with your actual network information.
