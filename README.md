# Backend: befriend.app

## DB Schema

https://drawsql.app/teams/befriend/diagrams/befriend-backend

### Setup

`git clone https://github.com/befriend-app/befriend-backend.git`

`cd befriend-backend`

`npm install`

`cp .env.example .env`

`node scripts/migrate.js`

`node server.js`
