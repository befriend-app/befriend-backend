https://drawsql.app/teams/befriend/diagrams/befriend-backend

### Setup

`git clone https://github.com/befriend-app/befriend-backend.git`

`cd befriend-backend`

`npm install`

`cp .env.example .env`

`node setup`

`node server.js`
